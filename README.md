# 📌 Project Overview

This project focuses on building an **interactive E-Commerce Sales Dashboard** in **Power BI** to monitor and analyze sales performance across multiple dimensions such as:

- Revenue
- Profit
- Quantity Sold
- Customer Analysis
- State-wise Sales
- Monthly Profit Trends
- Product Categories
- Payment Modes

The dashboard helps businesses make data-driven decisions by providing clear insights into customer purchasing behavior and product performance.

# 📈 Dashboard Features

## 🟣 Sales Analysis
- Total sales amount overview
- State-wise sales distribution
- Customer-wise sales contribution

- Monthly profit trend analysis
- Sub-category profit performance
- Profit comparison across products

## 🔵 Category Insights
- Quantity sold by category

- Product demand analysis

## 🟠 Payment Mode Analysis
- COD usage insights
- UPI transaction analysis

- EMI payment tracking

# 🚀 How to Use

1. Download or clone this repository
2. Open the `.pbit` file in Power BI Desktop
3. Load the CSV datasets when prompted
4. Refresh the dashboard
5. Explore insights using slicers and filters
